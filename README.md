# 🐾 Python 多功能桌寵 (DesktopPet)

## ✨ 特色功能

### 1. 智慧巡邏區域 (Smart Focus Zone)

為了確保開發者在寫程式或看影片時不被遮擋，桌寵具備智慧邊界邏輯：

- **活動限制**：桌寵被嚴格限制在螢幕右側 1/3 的區域活動。
- **開發禁區**：螢幕左側 2/3 被定義為工作禁區，桌寵永遠不會主動跨越。
- **固定啟動**：桌寵每次啟動都會出現在畫面右上方的預設位置，方便您快速找到它。

### 2. 系統工具箱

- 💻 硬體監控中心：即時 CPU、RAM 及網速顯示。點擊 RAM 數據可查看前 10 名資源佔用進程。
- 🌐 中英翻譯工具：支援自動語言偵測，快速進行中英互譯。
- 📊 螢幕使用紀錄：自動統計各程式使用時數，提供週趨勢圖（`Code` 會自動轉換為 `vscode`）。
- 💤 沉浸模式：支援最小化至系統匣（隱藏於工具列向上箭頭內），點擊圖示即可恢復。

## 📦 安裝需求

請開啟終端機（cmd 或 PowerShell），依序執行以下指令安裝必要套件：

- `Pillow` (圖片處理)

```powershell
python -m pip install pillow
```

- psutil (系統資訊抓取)

```PowerShell
python -m pip install psutil
```

- pystray (系統匣功能)

```PowerShell
python -m pip install pystray
```

- deep-translator (翻譯引擎)

```PowerShell
python -m pip install deep-translator
```

## 🚀 執行程式

1. 請開啟您的編輯器（如 VS Code）或終端機。
2. 切換路徑至專案資料夾後，在 `main.py` 的 terminal 輸入以下指令啟動桌寵：

```powershell
python main.py
```

## 📖 完整操作說明

- **左鍵按住拖曳**：可自由移動桌寵至螢幕右側 1/3 內的任何位置。
- **左鍵雙擊**：立即關閉桌寵主程式。
- **右鍵點擊**：彈出功能選單，可開啟硬體查詢、翻譯工具、使用統計或最小化桌寵。
- **恢復顯示**：若桌寵已最小化，點擊螢幕右下角工作列的「向上三角形」圖示，找到桌寵圖示並點擊即可喚回。

## ⚙️ 如何讓桌寵開機自動執行

這會讓桌寵與螢幕使用時間監控在每次開機時自動開啟：

1. **建立啟動檔**：在專案根目錄下建立一個文字檔，命名為 `start_pet.bat`。
2. **寫入路徑**：寫入以下內容（請將路徑改為您電腦的實際位置）

```batch
@echo off
cd /d "C:\您的路徑\deskpet"
start "" pythonw.exe main.py
```

## 🖼️ 自定義外觀 (Customization)

本專案預設使用提供的 `pet.png`。若想更換為你喜歡的角色：

1.  **準備圖片**：準備一張 **透明背景** 的 `PNG` 圖片。
2.  **覆蓋檔案**：將新圖片命名為 `pet.png`，並覆蓋原專案資料夾中的同名檔案。
3.  **縮放說明**：程式啟動時，會自動以原圖 **30%** 的比例進行縮放顯示。

## 🗑️ 如何完全刪除桌寵與紀錄數據

若要徹底從系統中移除程式，請執行以下三個步驟：

### 1. 強制結束程式

- 開啟 **工作管理員** (`Ctrl` + `Shift` + `Esc`)，手動結束所有名為 `Python` 或 `pythonw.exe` 的程序。
- 或者，在 **終端機 (cmd/PowerShell)** 輸入以下指令強制停止：

```bash
taskkill /F /IM pythonw.exe
```

### 2. 移除啟動捷徑

- 按下 `Win` + `R` 鍵，輸入 `shell:startup` 並按確定。
- 在開啟的資料夾中，找到並刪除 **`start_pet.bat - 捷徑`**。

### 3. 清空資料夾

- 直接刪除整個 `deskpet` 專案資料夾。
- **注意**：這將一併刪除您的歷史使用紀錄（路徑：`tools/screentime.json`），且資料刪除後無法復原。
